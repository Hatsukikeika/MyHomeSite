Handwrite AI OCR

This is an ORC tool for extracting the words/sentences to text. It was developed on Python 3.6, the main libraries used in this project are TensorFlow, PyQt, and Numpy. This tool allows users to open an image from the device or write down something by using a mouse on its whiteboard.
Words in the input will be captured and sliced into single letters through Image processing techniques and thresholding. After passing the letters into the ML model, the console will display the proper result.

My Game Engine

A simple game engine which builds from the ground up, using a combination of inheritance & composition. The libraries I used are ZMQ and SFML, and this project is developed in C++.
It is simple but comprehensive. It covers Event Systems, Time clock, Replay, External Script Support, and Multiplayer Support. Games created by this game engine include platformer games, bubble shooter, and snake game.
